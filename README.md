# adventofcode

Majority of util commands allow `--year` and `--day` flags to specify the year and day of the puzzle.

To init and fetch:

```bash
bun fetch
```

To run the sample tests:

```bash
bun runner
```

To run the full tests:

```bash
bun runner-full
```
